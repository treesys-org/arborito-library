# If and switch

**`if`/`elseif`/`else`** for general branching. **`switch`** can simplify multiple string cases and supports **`-Regex`** / **`-Wildcard`**.

@section: switch default

Use **`default`** for the fallback case.

@section: Next step

Pair with **`try/catch`** for terminating errors (next lesson).
