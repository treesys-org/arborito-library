# Spring Boot: starters, autoconfiguration, profiles

**Starters** bundle dependencies; **autoconfiguration** wires common beans when libraries are present.

Use **profiles** for environment-specific settings. Secure **Actuator** endpoints in production.
@quiz: Where does Spring Boot commonly store default configuration?
@option: mandatory web.xml
@correct: application.properties or application.yml
@option: env vars only
