# I/O: NIO.2, Paths, and safe serialization habits

**NIO.2** provides `Path`, `Files`, and atomic operations when the OS supports them.

```java
Path p = Path.of("data", "cfg.json");
String text = Files.readString(p, StandardCharsets.UTF_8);
```

@section: Java serialization

Untrusted `ObjectInputStream` data is a **remote code execution** risk via gadget chains. Prefer schema-driven formats.

@section: Pitfalls

* Reading huge files fully into RAM.
* Ignoring charset mismatches.
