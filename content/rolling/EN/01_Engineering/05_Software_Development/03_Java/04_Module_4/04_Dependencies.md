# Maven/Gradle dependencies

Maven **POMs** coordinate dependencies via GAV coordinates. Use **BOMs** / `dependencyManagement` to align versions and avoid classpath conflicts.
@quiz: Which file declares a Maven project’s dependencies?
@option: build.gradle.kts always
@correct: pom.xml
@option: package.json
