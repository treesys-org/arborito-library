# Variables: let, mut y shadowing

`let x = 5;` es inmutable; `let mut y = 0;` permite asignación. **Shadowing** redeclara nombre con nuevo tipo posible. Tipos inferidos salvo anotaciones necesarias.

@quiz: ¿Cómo se marca una variable mutable?
@option: volatile
@correct: let mut
@option: mut let
