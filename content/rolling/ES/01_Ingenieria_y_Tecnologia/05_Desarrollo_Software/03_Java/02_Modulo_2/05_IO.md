# Entrada/salida: NIO.2, Paths y serialización prudente

**NIO.2** (`java.nio.file`) unifica rutas portables y operaciones atómicas donde el SO lo permite.

```java
Path p = Path.of("data", "cfg.json");
String text = Files.readString(p, StandardCharsets.UTF_8);
Files.writeString(p, text, StandardCharsets.UTF_8,
    StandardOpenOption.CREATE, StandardOpenOption.TRUNCATE_EXISTING);
```

@section: Serialización Java

`ObjectInputStream` sobre datos no confiables es vector de **gadget chains** (deserialización insegura). Prefiere JSON/MessagePack con esquemas o formatos explícitos.

@section: Errores frecuentes

* Asumir que `File.separator` basta para normalizar rutas Unicode raras.
* Leer ficheros gigantes enteros en memoria sin streaming.
